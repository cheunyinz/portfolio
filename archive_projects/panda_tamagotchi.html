Panda tamagotchi

03-2021

For this project we had to create a mini game in the form of a tamagotchi. I chose to make a panda tamagotchi. As a visitor you can interact with the panda and the panda in the other hand can talk back under certain conditions.

At the beginning we had to come up with an idea. After that we had to think about different interaction we could create. Once we had that we started coding.

When we started to code it was quiet a challenge since it was my first time using Javascript. Luckily with the help of my teachers and the internet I managed to create the mini game.

In the top right corner of the website you can see the status bar. The first one is the happy bar and the second one the hunger bar. Both status bars decrease every once a while.

Once the hunger or happy bar is empty the panda will let you know he is unhappy or hungry. To get the panda happy you can press the W button to let him jump. If the panda is hungry you can drag bamboo to him.

During this course I learned the basics of Javascript, for example boolean, if else statements, key events, intervals, and more.
