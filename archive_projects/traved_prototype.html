TRAVED. prototype

01-2021

For this project I had to create a mobile web app prototype for the NOTED. book using HTML and CSS. The NOTED. book is a guide for international students in Amsterdam. The mobile web app had to add value to the book to make the stay for the international students even better.

The project started with coming up with a concept. After some brainstorming I decided to create a mobile web app where students can create day trips where after other students can join in if they are interested. Once a student join in, there would be a chat function where they can communicate with each other. This way international students could meet new people while discovering the Netherlands.

After I had a concept I had to make different wireframe and color schemes, and I had to design a Hi-Fi prototype. Once I had the Hi-Fi prototype I started coding.

This was the first time I started using HTML and CSS. At first I had trouble with CSS, especially with positioning elements. Instead of using flexbox, grid, and margin I used position absolute, pixels, and inset properties. This made the postioning of elements much harder since I had to position each element individually. Luckily I quickly learned to use flexbox and margin to position elements.

During this project I learned how to write semantic HTML and I learned the basics of CSS, for example flex box and box model.
