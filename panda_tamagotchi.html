Panda tamagotchi

For this project we had to create a mini game in the form of a tamagotchi. I chose to make a panda tamagotchi which you could interact with. As a visitor you can give him food and let him jump up. The panda in the other hand can talk back under certain conditions.

In the start we had to come up with an idea. After that we had to think about different interaction we could create. Once we had that we started coding.

This project was the first time I used Javascript. During the lessons we learned the basics of Javscript.
